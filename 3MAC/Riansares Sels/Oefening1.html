Programmeertaal is een formele taal waarin geschreven staat welke opdrachten een computer moet uitvoeren. Enkele voorbeelden van deze programmeertaal zijn dus: html, CS6? (C++), Java.

Een scripttaal is een programmeertaal die scripts schrijft. Scripts zijn kleine programma’s die veelvoorkomende taken automatiseren. Het kan ook gebruikt worden om grote eenmalige taken te verrichten. De meeste applicaties bevatten een scripttaal.

Java is een objectgeoriënteerde programmeertaal. Deze is grotendeels gebaseerd op de C++ programmeertaal maar is uitgebreider.

JavaScript is een scripttaal die webpagina’s interactief maakt en webapplicaties ontwikkelt.
