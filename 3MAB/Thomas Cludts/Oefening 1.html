Coding with HTML & CSS

In deze blogpost zal het voornamelijk gaan over het coderen en het opmaken van een eigen website. Het is als wijze van toepassing om kennis te maken met de wereld van HTML en CSS. Het is belangrijk dat je de verschillende tags kent, maar nog belangrijker is dat je moet weten hoe je de tags moet gebruiken in de praktijk.

HTML

HTML staat voor HyperText Markup Language. Het is een "taal" waarin documenten geschreven worden die gepubliceerd zullen worden op het internet. We kunnen dus eigenlijk spreken van webdocumenten. Zoals we al weten bestaat HTML uit platte tekst. Deze platte tekst krijgt bepaalde tags zodat de browsers weten hoe ze dit document moeten interpreteren.

Er zijn al heel wat versies van HTML verschenen, de laatste dateert van 2001 met HTML 4.01. In december 2012 is er een nieuwe versie aanbevolen om te gaan gebruiken.

CSS

CSS staat voor Cascading Style Sheets. CSS wordt gebruikt voor het vormgeven van webpagina's en webdocumenten. De CSS-Informatie wordt in het HTML-bestand uitgeschreven, zo weten browsers ook dat deze bij elkaar horen. Het is wel mogelijk om ook je CSS-informatie in een apart bestand te zetten en er dan naar te verwijzen. Dit aparte bestand wordt een Stylesheet genoemd.

Tags

Hieronder zullen een aantal voorbeeldtags uitgelegd worden (ik ga het vooral hebben over enkele nieuwe, handige tags). Het is belangrijk om tags goed onder de knie te krijgen, zo kan je een website nog beter uitwerken. De basis is enorm belangrijk, maar HTML en CSS gaat veel verder. Een belangrijke website om tags te raadplegen is W3Schools.

Tabellen

Hieronder staat de werking van tabellen. Tabellen open je met de tag "table". Een rij maak je met "tr". "th" en "td" staan voor respectievelijk vetgedrukte en normale woorden.

Details

Tussen de tag van "Details" kan je extra informatie plaatsen. Deze informatie wordt pas weergegeven op de website wanneer er op het pijltje geklikt wordt. Een voorbeeld hiervan kan je vinden onderaan deze webpagina.

HR/

Aan de hand van de "HR-tag" kan je een einde van een onderdeel van je website afsluiten. Dit gebeurt door middel van een lijn. Deze lijn wordt getrokken op de plaats waar je de tag invoert. (Zie voorbeeld hieronder!)

Embedden van nieuwe elementen

Met de tag "embed" kan je nieuwe elementen toevoegen, zoals audio-bestanden of videobeelden. De browser waarin je de webpagina bekijkt moet wel beschikken over de juiste plug-ins om deze bestanden te kunnen bekijken. In het voorbeeld hieronder maak ik gebruik van een gewone afbeelding, om zo ook aan te tonen dat je dus niet per se gebruik moet maken van de "IMG-tag"

Footer

Deze tag zorgt ervoor dat gegevens onderaan je pagina verschijnen. Je kan het zien als de kleine lettertjes van een webdocument.

Andere tags?

Er zijn natuurlijk nog tal van andere tags. Maar het doel van deze webpagina is niet om te copy pasten, maar eerder om meer bekend te geraken met de werking ervan. Ik ga dus niet alle tags bespreken, want dat hebben andere sites al eerder gedaan.
