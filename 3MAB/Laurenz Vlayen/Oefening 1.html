The internet, what else?

Het internet is meer dan enkel browsers, applicaties en websites, het is een systeem om computers met elkaar te verbinden via interconnected computer networks. Vervolgens zijn er nog protocollen, dit zijn afspraken in verband met wie wat kan doen op je computer. Om te kunnen zien wie wie is op het internet, heb je een IP-adres. Dit adres is geen nummer van een pc, maar het is de nummer van de host die je van internet voorziet. Omdat het internet niet gecentraliseerd is, kan men stellen dat niemand de baas is. Je bent enkel baas over je eigen machine. In tegenstelling tot internet, is een intranet wel gecentraliseerd, niet iedereen kan zomaar toetreden tot een bepaald intranet. Je moet je eerst aanmelden op het lokaal intranet. Internet is ontstaan nadat het intranet meerdere bedrijven via telefoonlijnen werd verbonden. Een host (bv. Telenet) moet deze verbindingen wel leggen, dus in de praktijk heb je deze schakels nodig om internet te kunnen gebruiken. Bij opsporing van een crimineel moet politie naar deze host gaan om te vragen wie van de klanten op dat ogenblik een bepaalde data verstuurde. Internet Corporation for Assigned Names and Numbers ofwel ICANN regelt de toekenning van de IP-adressen over heel de wereld. De Domain Name System (DNS) zorgt ervoor dat moeilijke IP-adressen van websites worden omgezet naar gemakkelijker te onthouden websites, bv. www.google.com . Het internet ‘hangt vast’ op een tiental ‘supernotes’ over heel de wereld. Moest één van deze ‘supernotes’ crashen zou er een heel deel van de wereld even niet meer op het internet kunnen.

Intermezzo

In tijden van oorlog zoals bijvoorbeeld in Syrië, waar de regering het internet heeft afgesloten, kan men gebruik maken van een Ad Hoc Network. Dit is een netwerk (bv. Bluetooth) waardoor er een draadloos internet kan worden opgezet en dit zolang je je in de zone van het netwerk bevindt. In Syrië gaan op een bepaald tijdstip verschillende drones de lucht in die een gsm signaal uitzenden. Via dit signaal kan met gebruik maken van het lokaal netwerk en kan men bijvoorbeeld skypen.

Datatransfer

Er zijn twee grote begrippen die betrekking hebben tot datatransfer. Aan de ene kant heb je File Transfer Protocol (FTP), die ervoor zorgen dat grote documenen gemakkelijker kunnen verzonden worden via het internet. Aan de andere kant zijn er application programming interfaces. Een API is een verzameling definities op basis waarvan een computerprogramma kan communiceren met een ander programma of onderdeel (meestal in de vorm van bibliotheken). Vaak vormen API's de scheiding tussen verschillende lagen van abstractie, zodat applicaties op een hoog niveau van abstractie kunnen werken en het minder abstracte werk uitbesteden aan andere programma's.

Communicatie over het internet

Over het internet kan je via allerhande programma’s communiceren met anderen. Zo is er Instant Messaging (IM) dat er voor zorgt dat mensen met elkaar kunnen communiceren aan de hand van getypte tekst. Via een Voice-over Internet Protocol (VoIP) kan je (video)gesprekken voeren met mensen, een voorbeeld hiervan is Skype.

Een overload aan begrippen

  • Inter-linked hypertext documents: Functie die teksten aan elkaar linkt via andere links.
  • Peer-to-peer networks (p2p): Een op infrastructuur van het internet gecreëerd eigen intranet, maar dit is illegaal. (bv. The Pirate Bay)
  • Webbrowser: De browser leest de html-codes van de website, maar heeft ook verschillende applicaties en functies zoals bijvoorbeeld ‘History’ of ‘Favorites’.
  • Rendering engines: Dit is een stukje software dat html codes kan lezen en weergeven op het scherm.
  • Balkanization: Landen die niet willen dat Amerika hun land controleert, gaan een onafhankelijk internet organiseren.
  • Balloon internet: Ballonnen met een internetzender die door de natuurlijke flow wordt meegedragen.
